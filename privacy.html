개인정보처리방침

서비스는 기본적으로 개인식별정보 수집을 목적으로 하지 않습니다. 사용자 입력값 중 Bybit API Key/Secret은 모니터링 세션 구동 목적에 한해 메모리에서 사용됩니다.

  • API Secret은 화면에 마스킹 표시만 허용
  • 거래 기록 CSV에는 주문/손익 정보만 저장
  • 운영자는 키 원문 저장/전송을 금지
